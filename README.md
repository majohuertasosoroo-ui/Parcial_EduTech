#  Solución Parcial Corte 2: Plataforma Edu-Tech

*Integrantes:* Alessandro Di Mauro Bonilla Correa, Gabriela Ruiz Pérez, María José Huertas Osorio.

##  Arquitectura Implementada
1. *Limpieza con Pandas:* Extracción de datos sucios (CSVs), normalización 1NF (separación de campos compuestos) e imputación de correos y edades vacías.
2. *Migración SQLite:* Creación del archivo parcial_edutech.db con relaciones PK/FK activas mediante PRAGMA.
3. *Módulo Inteligente (POO):* Implementación de la clase PlataformaEduTech ejecutando operaciones CRUD (Inscripciones) y reportes multicruzados (JOIN) entre estudiantes, cursos y profesores.
